# 广元三天两晚旅游攻略

> **节奏定位：** 轻松游，不赶点，每处留足时间。  
> **核心原则：** 不爬剑门关；凤凰山若封闭跳过；昭化交通优先**网约车直达**（省去等班车压力）。  
> 所有车次、班次以出行当日 **12306 / 高德地图 / 广运神马小程序 / 景区公众号** 为准。

### 晚间二选一（出发前定好方案）

#### 方案 A：回广元吃鱼 + 嘉陵江游船

适合：想在市区享受江景夜晚的人。

```
昭化  →（网约车回广元南河，约 18:00～19:00）
     →  南河江边鱼庄晚餐（上海路 / 滨河南路，约 19:00～20:30）
     →  嘉陵江夜游游船（按船票时间到码头）
     →  下船后沿滨江散步 → 回酒店
```

> 游船在美团 / 携程 / 抖音搜「广元 嘉陵江 游船」提前购票；注意停航公告；提前 20 分钟到码头。

#### 方案 B：留在昭化看夜景（省力省钱）

适合：不想赶路、想体验古城夜晚的人。

```
昭化古城  →  黄昏留在古城（灯笼街、西市渐入夜色）
          →  古城内晚餐（面馆、农家菜）
          →  《葭萌春秋》演出（以景区当日公告为准）
          →  22:00 前后网约车回广元（约 50～90 元）
             或直接住昭化（次日从容回市区）
```

## 五、周日：明月峡 → 午餐 → 广元站

**整体节奏：上午出发游览，午餐后收尾，下午从容返站。**

### 推荐：明月峡历史文化旅游区（主推，不爬山）

- **位置：** 朝天区，嘉陵江峡谷古栈道文化
- **怎么去：** 酒店网约车直达，约 40～55 分钟，单程约 60～100 元
- **游玩：** 沿江栈道步行，五条古道并行峡谷，轻松不爬山，约 2.5 小时

```
09:00  酒店出发（叫好网约车）
09:50  到达明月峡游客中心
10:00～12:30  游览明月峡（沿栈道慢走，拍照，不赶）
12:30  叫车离开
13:00～14:00  午餐（朝天城区简餐，或返回市区再选）
14:30  出发前往广元站
15:00  到达广元站（进站安检）
```
